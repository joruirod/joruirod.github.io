How to use the plugin

Plugin

  • Clean and lightweight Markup

    • An unordered list inside an id slider
    • Optional arrows inside a class called controls

  • Flexible number of images

    • Unlimited images inside the unordered list

  • Different sizes supported

    • The size of the images can be different (the smaller will be taken)
<div id="slider">
	<ul>
		<li><img src="demo/img/001.jpg"  /></li>
		<li><img src="demo/img/002.jpg"  /></li>
		<li><img src="demo/img/003.jpg"  /></li>
	</ul>
</div>

<div class="controls">
	<span id="left"><</span>
	<span id="right">></span>
</div>

  • Few slider styles

    • Do not forget to link the aSlider.css

    • Mandatory jQuery

      • It works fine for 1.5+

    • Lightweight

      • Link the aSlider.js at the bottom of the body

<script src="stylesheets/aSlider.min.css">



<script src="js/jquery.min.js"></script>



<script src="js/aSlider.js"></script>`
